# PhotoAlbum
This program allows users to list and view photos.

To run the program, run:
```
python manager.py
```

# Configuration
The file "dogs.txt" contains the listing of photos and their info.

"photo.py" and "album.py" represent a photo and album class, respectively.

"manager.py" contains the code to interact with the user.

"graphics.py" is a graphics library that should work on different platforms.
